  • Font-size: the font size is what depicts the size of the text that the writer wants to be portrayed, the greater the "px" the larger the font becomes.
  • Font-family: the font family functions is a specific font family (i.e. Arial, Times New Roman etc..) that changes the font family of the text that is trying to be portayed.
  • Color: the color function is what changes the color of the text.
  • Line-height: the spacing between each line of text. The greater the line height, the greater the space there is in between each line.
  • Padding-left: the amount of space that is wanted around a specific amount of content to the left.
  • Padding-right: the amount of space that is wanted around a specific amount of content to the right
  • Padding-top: the amount of space that is wanted around a specific amount of content to the top.
  • Font-weight: controls how bold or how light the text is being portrayed.
  • Padding: the desired amount of space that is wanted around content.
  • Float: the float property specifies whether ornot a box (an element) should float.
  • Text-decoration: different types of text decoration have different functions (underline, line through, overline, inherit) that changes the way that the text is displayed.
  • Display: the display property specifies the type of box used for an HTML element.
  • Width: sets the width of an element
  • Height sets the height of an element
  • Text-align aligns where the desired text is to be displayed (center, left, right)
  • Font-style: specicies the font style for a text (normal, italic, oblique)

    Defining the p class style rule:

    The .class selector selects elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the name of the class. You can also specify that only specific HTML elements should be affected by a class. To do this, start with the element name, then write the period (.) character, followed by the name of the class. An example of the use of the html sytnax using the class style rule is:
    (p class="center large">This paragraph refers to two classes. (/p)

    The "id" attribute allows authors to assign a unique name to an element. CSS2 allows authors to specify style rules that apply to a single instance of an element, based on its "id" value.

    To match an element with a given "id" value, the selector must contain "#" followed by the "id" value. In the following example, the style rule contains no selector information and therefore matches any element that has id="z98y". The rule will thus match for the P element:


    (HEAD)
    (STYLE)
    #z98y {letter-spacing: 0.3em }
    (/STYLE)
    (/HEAD)
    (BODY)
    (P id=z98y>Wide text(/P>=)
    (/BODY)

    In the next example, however, the style rule will only match an H1 element that has id="z98y". The rule will not match the P element in this example:

    (HEAD)
    (STYLE)
    H1#z98y { letter-spacing: 0.5em }
    (/STYLE)
    (/HEAD)
    (BODY)
    (P id=z98y>Wide text(/P)
    (/BODY)
