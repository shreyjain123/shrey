Question and Answer

1) How would you center everything on an entire page?

To center everything on the entire page, one would use the CSS function "margin". This way everything would stay in the center of the page. They could also use the padding function to ensure that there is an adamant amount of space around the content that they are writing. The last way of centering everything on the entire page is to use the "text-align" function would align all of the text on a page in the center.


2) How would indent a single word and put a square bullet in front of it?

To indent a signle word and put a bullet in front of it, one would use the "text-indent" function to make the single word indented a specific amount. To put a square bullet below the single word, use the "list-style-type: square;" function in the CSS portion of your html file. Once done that, you can use the "(li)" function to create a list below the word that was orinially written.
