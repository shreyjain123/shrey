What is PHP?

  • PHP stands for PHP Hypertext Preprocessor
  • PHP is a server-side scripting language, like ASP
  • PHP scripts are executed on the "server"
  • PHP supports many databases:
  • MySQL
  • Informix
  • Oracle
  • Sybase
  • Solid
  • PostgreSQL
  • Generic ODBC,etc.
